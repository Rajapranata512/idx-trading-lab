Safety notes

Public deploy posture

  • Serve the Python app behind a reverse proxy and bind it to 127.0.0.1.
  • Set IDX_WEB_USERNAME and IDX_WEB_PASSWORD before public exposure.
  • Keep the premium dashboard public, but reserve operational routes for authenticated operators.
  • Use localhost to trigger run-daily so the pipeline cannot be kicked off remotely.
